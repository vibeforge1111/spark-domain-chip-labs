# Scoring Rubric

Score each run from 0 to 2 per dimension.

| Dimension | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Delete integrity | Misses partial writes | Flags partial writes without counts | Flags request id, expected, requested, and accepted counts |
| Active-state grounding | Blends trace and active state | Shows active state but unclear authority | Separates active profile, topic trace, and recommendation |
| Telegram actionability | No natural route | Routes but reply is noisy | Routes and returns concise status plus next step |
| Authority boundary | Repairs automatically | Mixed wording | Diagnosis-only boundary is explicit |
