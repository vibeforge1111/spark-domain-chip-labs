# Startup YC Scoring Rubric

Source repos:

- `specialization-path-startup-yc`
- `domain-chip-startup-yc`
- `startup-bench`

Rubric dimensions for this fixture:

- Doctrine alignment.
- Operational specificity.
- Risk management.
- Revenue quality.
- Trap resistance.
- Fresh-agent absorption delta.

Promotion requires no-pack vs pack vs validated-pack comparison. Score improvements that come from changed benchmark weights, changed expected outcomes, or language inflation are not valid.
