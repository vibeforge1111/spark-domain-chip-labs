# Startup YC Doctrine Reference

Source repo: `domain-chip-startup-yc`

Core doctrine primitives used by this fixture:

- Make something people want.
- Retention beats acquisition.
- Fundraising is not validation.
- Do things that do not scale.
- Do not hire before product-market fit.
- Default alive beats default dead.
- Revenue quality matters more than vanity metrics.

The fixture points to the live domain chip rather than duplicating the full doctrine corpus.
